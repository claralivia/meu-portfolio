# Portfólio de Desenvolvedora Fullstack | Clara Lívia

Portfólio profissional moderno, com foco em experiência do usuário e código limpo, desenvolvido para destacar minha experiência como Desenvolvedora Fullstack Pleno, minha paixão por design (Glassmorphism, Dark Mode) e minha stack em Vue.js e Node.js.

## Stack & Arquitetura

Este projeto foi construído com uma arquitetura de frontend de alto desempenho:

- **Framework:** Vue.js 3 (Composition API)
- **Tooling:** Vite
- **Estilização:** Tailwind CSS (com foco em Glassmorphism e Dark Mode)
- **Linguagem:** TypeScript / JavaScript

## Visual & Funcionalidades

O design segue uma estética minimalista e elegante, com funcionalidades de UX aprimoradas:

- Navbar Sticky e Transparente (Efeito Vidro)
- Rolagem Suave (Smooth Scroll)
- Integração reativa com o mouse (Efeito Blob de fundo)
